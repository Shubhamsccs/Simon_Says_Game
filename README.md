# JavaScript Mini Games 🎮

A collection of interactive web games built to practice and solidify my understanding of JavaScript logic, state management, and DOM manipulation. 

## 🚀 Featured Game: Simon Game
A classic memory game where the user must remember and repeat an increasingly long sequence of colors. 

## 🛠️ Tech Stack
* **HTML5:** Game structure.
* **CSS3:** Styling and animations (button flashes).
* **JavaScript:** The core engine of the project! Handles event listeners, arrays to store sequences, user input validation, and level progression.

## 💡 What I Learned
This project was a massive leap from static web pages. It helped me master:
* Selecting and manipulating DOM elements.
* Using `setTimeout` for delays and animations.
* Handling complex conditional logic and user interactions.

## ⚙️ How to Play
1. Clone this repository.
2. Open `minigames.html` in your web browser.
3. Press any key to start the Simon Game and test your memory!
